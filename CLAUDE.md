# code_learing

Learning project for code documentation and tooling.

## Skills

### /thesis2ppt
Generate a graduation defense PPT from a thesis document (PDF, DOCX, TXT, MD).

See `.claude/skills/thesis2ppt.md` for full documentation.

Core script: `scripts/thesis2ppt.py`
Style config: `templates/academic_blue.json`

## Setup

```bash
pip install -r requirements.txt
```
